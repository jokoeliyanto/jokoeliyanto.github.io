Matplotlib & Seaborn

  • Matplotlib adalah salah satu module dalam pemrograman Python yang digunakan untuk visualisasi data dalam bentuk graph.
  • Graph menggunakan Matplotlib digambarkan pada suuatu figure, yang di dalamnya terdiri dari satu atau lebih axes.
  • Seaborn adalah pustaka visualisasi data Python yang dikembangkan dari matplotlib. Library ini menyediakan antarmuka tingkat tinggi untuk menggambar grafik statistik yang menarik dan informatif.

Komponen dari sebuah Matplotlib Figure

  • Figure adalah media dasar untuk melakukan visualisasi menggunakan Matplotlib
  • Dalam sebuah figure kita dapat memiliki beberapa axes
  • axes adalah area gambar dalam sebuah figure
  • axes berada di dalam figure
  • axis sumbu dalam dimensi visualisasi data. Jumlah axis bisa dua sampai tiga yaitu x, y ,z

Berikut adalah beberapa cara untuk membuat figure dan axes:

# Membuat sebuah figure tanpa axes
fig = plt.figure()
<Figure size 432x288 with 0 Axes>
# Membuat sebuah figur dengan sebuah axes
fig, ax = plt.subplots()

# Membuat sebuah figure dengan 2(baris)x3(kolom)[6 axes] grid axes
fig, axs= plt.subplots(2,3)

def single_line_plot(df, title, size):
  '''
  Fungsi line_plot_sederhana(df) digunakan untuk melakukan plot line sederhana
  Input:
        df    = dataframe input dengan kolom data dan kategori
        judul = judul gambar
        size  = ukuran gambar
  '''

  # Mengatur style plot dengan grid
  sns.set_style("whitegrid")

  # Data sampel
  x = np.array(trend_df.index)
  y = np.array(trend_df.nomor_registrasi)

  # Membuat sebuah figure dan sebuah axes

  # Mengganti label sumbu y

  # Menampilkan hasil plot

Membuat fungsi bar plot sederhana

def bar_plot_sederhana(df, title, size):
  '''
  Fungsi bar_plot_sederhana(df) digunakan untuk melakukan plot bar sederhana
  Input:
        df    = dataframe input dengan kolom data dan kategori
        judul = judul gambar
        size  = ukuran gambar
  '''

  # Memilih tema plot
  sns.set_theme(style="whitegrid")

  # Membuat sebuah figure dan sebuah axes
  fig, ax=plt.subplots(figsize=size) # figsize=(x,y) untuk menentukan ukuran gambar

  # Melakukan bar plot
  ax = sns.barplot(data=data_profit,      # Nama DataFrame
             y="profit",                # Data sumbu Y
             x="spesialisasi",          # Data sumbu X
             palette="Blues_d"          # Palete warna
            )


  # Mengatur sudut rotasi label stepsize sumbu x
  plt.xticks(rotation=80)

  # Mengganti label sumbu x
  ax.set_xlabel('Spesialisasi')

  # Mengganti label sumbu y
  ax.set_ylabel('Profit (Juta Rp)')

  # Mengatur Judul Gambar
  ax.set_title(title)

  # Menampilkan Gambar
  plt.show()
bar_plot_sederhana(data_profit, title='Judul Gambar Pertama', size=(10,5))

# import libraries
import seaborn as sns
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.patches as mpatches

# load dataset
tips = sns.load_dataset("tips")

# set plot style: grey grid in the background:
sns.set(style="darkgrid")

# set the figure size
plt.figure(figsize=(14, 14))

# top bar -> sum all values(smoker=No and smoker=Yes) to find y position of the bars
total = tips.groupby('day')['total_bill'].sum().reset_index()

# bar chart 1 -> top bars (group of 'smoker=No')
bar1 = sns.barplot(x="day",  y="total_bill", data=total, color='darkblue')

# bottom bar ->  take only smoker=Yes values from the data
smoker = tips[tips.smoker=='Yes']

# bar chart 2 -> bottom bars (group of 'smoker=Yes')
bar2 = sns.barplot(x="day", y="total_bill", data=smoker, estimator=sum, ci=None,  color='lightblue')

# add legend
top_bar = mpatches.Patch(color='darkblue', label='smoker = No')
bottom_bar = mpatches.Patch(color='lightblue', label='smoker = Yes')
plt.legend(handles=[top_bar, bottom_bar])

# show the graph
plt.show()
